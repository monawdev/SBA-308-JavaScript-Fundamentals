# SBA-308-JavaScript-Fundamentals
JavaScript application that processes learner assignment data, calculates weighted averages, applies late penalties, and demonstrates core JavaScript fundamentals including functions, loops, arrays, objects, and error handling.



## Features
- Weighted averages
- Late submission penalties
- Due date filtering
- Error handling
- Array and object manipulation

## Technologies Used
- JavaScript
- Node.js

## How to Run

1. Clone the repository
2. Open terminal
3. Run:

```bash
node index.js
